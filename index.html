Visual Molecular Docking Simulator

Watch Molecular Recognition Happen

This is an interactive 3D science workspace that plays molecular docking as a trajectory movie. Instead of showing only the final pose, the simulator calculates and animates the full physics of the search: random search restarts, steric clash rejections, H-bond formations, and local optimizations.

🖥️ Run Locally & Upload Custom Files

To run the full dynamic application—including support for uploading custom PDB proteins and SDF/MOL2 ligands with real-time pocket detection and interactive simulation—run the app on your computer using Streamlit:

# Install scientific dependencies pip install -r requirements.txt # Run the Streamlit application streamlit run app.py
